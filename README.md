# 📊 Udemy Courses Finding Analysis  

## 📝 Project Overview  
This project analyzes Udemy course data to uncover insights into **course trends, pricing, revenue, and subscriber behavior**. Using **Power BI**, we visualize key performance indicators (KPIs) to understand the **most popular subjects, pricing structures, and sales trends**.  

---

## 📂 Dataset Summary  
The dataset covers Udemy course offerings across multiple **years, subjects, and pricing tiers**, helping to identify **high-revenue courses, popular subjects, and pricing impacts on subscriber growth**.  

### **Key Data Points**  
| **Metric**                  | **Value / Insight** |
|-----------------------------|---------------------|
| 📚 **Total Courses**        | *1199* |
| 👥 **Total Subscribers**    | *8 Million* |
| 💰 **Total Revenue**        | *$628 Million* |
| ⏳ **Total Course Duration** | *6,710 Hours* |
| 🆓 **Free vs Paid Courses** | *88.91% Paid, 11.09% Free* |
| 🎓 **Course Levels** | All Levels, Beginner, Intermediate, Expert |

---

## 🎯 Key Insights & Findings  

### 📌 **Top Course Categories (By Subscribers & Revenue)**  
✅ **Web Development** - *Highest Revenue ($627.5M) & Subscribers (7.93M)*  
✅ **Business & Finance** - *Revenue ($123.7M) & Subscribers (1.86M)*  
✅ **Graphic Design** - *Revenue ($76.9M), Subscribers (1.06M)*  
✅ **Musical Instruments** - *Revenue ($53.3M), Subscribers (846K)*  

### 📌 **Course Pricing & Subscriber Trends**  
- **Premium courses** generate **$0.54B revenue**, while **low-priced courses** contribute only **$0.09B**.  
- **Beginner-level courses** attract the highest number of subscribers (**4.1M+**).  
- **Expert-level courses** have the highest average price (**$113.67** per course) but fewer subscribers.  

### 📌 **Most Popular Courses (By Subscribers)**  
1️⃣ **Learn HTML5 Programming From Scratch** - *269K subscribers*  
2️⃣ **Coding for Entrepreneurs Basic** - *161K subscribers*  
3️⃣ **The Web Developer Bootcamp** - *122K subscribers*  
4️⃣ **Build Your First Website in 1 Week** - *120K subscribers*  
5️⃣ **The Complete Web Developer Course 2.0** - *115K subscribers*
